# Release Notes: Representation Packet Technical Report v1.0.0

This release provides a public-safe Preprint / Technical Report documenting a
deterministic representation layer for finite, noisy sports trajectories. It
was released on 2026-07-13 and is not peer reviewed.

## Release identity

- Branch: `codex/representation-packet-technical-report-v1.0.0`
- Tag: `representation-packet-technical-report-v1.0.0`
- Version DOI: `10.5281/zenodo.21340626`

- Historical version DOI: `10.5281/zenodo.21193034`

## Included

- Technical-report manuscript in Markdown, DOCX, and PDF formats.
- Release citation metadata and release manifest.
- Public claim-boundary documents, limitations, and staged sensitivity
  summaries.
- Public-safe figure and table views.

## Not included

- Source trajectory CSV files.
- Canonical trajectory tables or descriptor tables.
- Primitive, topology, or graph artifacts.
- Overlay HTML or PNG files containing trajectory geometry.
- Projection origins, geographic coordinate details, source-data paths, or
  private implementation materials.

## Claim boundary

This release documents representation and interpretation boundaries under
explicit coordinate, threshold, tolerance, score, and staged sensitivity
assumptions. It does not claim external measurement accuracy, robust
detection, sport-specific semantic interpretation, calibrated probability
confidence, universal CRS enforcement, production deployment readiness, or
application-specific validation.
