# Public Figure and Table Views

This index provides GitHub-rendered, public-safe views of the figure and table material used by the Paper A v0.10 draft.

The private working packet retains trajectory-derived artifacts such as overlays, canonical tracks, descriptor tables, primitive candidate files, topology files, graph files, and coordinate-provenance JSON. This public repository exposes only manuscript-level summaries and restricted-asset notes.
