# Release Notes: v0.10.3-public-safe

This release provides a public-safe companion packet for Paper A manuscript review and is issued after the Zenodo GitHub webhook was confirmed active.

## Included

- Public-safe Paper A v0.10 manuscript draft.
- Public claim ledger, limitations, and candidate/tolerance doctrine.
- Public staged sensitivity evidence summary.
- GitHub-rendered figure and table views.
- Public-safe staged sensitivity CSV.
- Citation metadata through `CITATION.cff`.
- Repository license metadata.

## Not Included

- Source trajectory CSV files.
- Canonical trajectory tables.
- Descriptor tables.
- Primitive, topology, or graph artifacts.
- Overlay HTML or PNG files.
- Coordinate-provenance JSON files.
- Private source-code modules from the local working repository.

## Claim Boundary

This release supports manuscript-level review of a deterministic representation packet under explicit coordinate, threshold, tolerance, score, and staged sensitivity assumptions.

It does not support claims of external measurement accuracy, robust detection, sport-specific semantic interpretation, calibrated probability confidence, universal coordinate-system enforcement, or production deployment readiness.
