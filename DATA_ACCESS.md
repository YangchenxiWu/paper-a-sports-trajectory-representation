# Data Access

This public-safe repository does not include raw or generated trajectory artifacts.

The withheld artifacts include:

- source trajectory CSV files;
- canonical metric trajectory tables;
- descriptor tables;
- primitive candidate JSON files;
- topology candidate JSON files;
- symbolic graph JSON files;
- overlay HTML/PNG files;
- coordinate-provenance JSON files containing projection origins or geographic coordinate information.

These files are withheld because they may reveal trajectory geometry, source-data location, source-data layout, or location-derived metadata.

The public repository includes manuscript text, claim-boundary documentation, staged sensitivity summaries, and literature notes. Those materials are intended to make the representation claim and its limitations inspectable without publishing trajectory-level data.

Data and full packet artifacts may be made available only under an appropriate data-access arrangement.
