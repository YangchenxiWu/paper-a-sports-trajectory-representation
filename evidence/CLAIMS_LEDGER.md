# Public Claims Ledger

This public-safe ledger records the claims that Paper A is intended to support.
It omits private planning details and trajectory-level evidence paths.

## Supported Public Claims

| Claim | Public support | Boundary |
| --- | --- | --- |
| Packet-level coordinate contract | The manuscript describes explicit coordinate type, source CRS, projection method, axis convention, units, and metric assumptions. | Packet-level only; not universal CRS enforcement. |
| Deterministic descriptor generation | The manuscript describes deterministic per-point motion and geometry descriptors. | Descriptors are finite-observation quantities, not continuous physical measurements. |
| Threshold-conditioned primitive candidates | The manuscript reports configured thresholds and candidate terminology. | Candidates are not confirmed movement structures or sport-specific semantics. |
| Tolerance-conditioned topology candidates | The manuscript reports topology tolerance semantics and candidate-count interpretation. | Counts are not unique physical crossings or externally confirmed structures. |
| Symbolic graph and overlay artifacts | The manuscript describes generated graph and inspection overlay artifacts. | Public repository withholds trajectory-revealing artifacts. |
| Deterministic heuristic score semantics | The manuscript states that score fields are deterministic heuristic scores. | Not calibrated probability, not artifact-survival evidence, and not comparable statistical confidence across candidate types. |
| Staged sensitivity evidence | The public summary table reports staged sensitivity probes for sampling, smoothing, curvature, primitives, and topology. | Staged evidence is not a unified validation benchmark. |

## Unsupported Public Claims

Paper A does not claim:

- external measurement accuracy;
- robust detection;
- sport-specific semantic interpretation;
- production deployment readiness;
- universal CRS enforcement;
- calibrated probability confidence;
- population-level generalization across athletes, devices, sessions, or sports.

## Private Evidence Boundary

Some manuscript values are supported by private packet artifacts that are not included in this public repository because they may reveal trajectory geometry, source-data layout, or location-derived metadata. See `DATA_ACCESS.md` and `PUBLIC_EVIDENCE_INDEX.md`.
